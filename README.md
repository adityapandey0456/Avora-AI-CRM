# 🚀 Avora AI Healthcare CRM

Avora AI is an intelligent, AI-first Customer Relationship Management (CRM) platform specifically designed for Healthcare Professionals (HCPs) and medical representatives. It streamlines the logging of doctor interactions using advanced AI agents, ensuring professional follow-ups and strict compliance monitoring.

## 🌟 Key Features

*   **AI-Powered Agent (LangGraph):** Uses a sophisticated multi-tool agent to process interaction logs, extract action items, and analyze sentiment.
*   **Voice-to-Text Logging:** Integrated browser-based speech recognition for hands-free interaction logging.
*   **Compliance & Safety:** Automatically flags potential adverse events, off-label usage, or compliance risks in doctor discussions.
*   **Smart Email Drafting:** Generates professional, context-aware follow-up emails only when explicitly requested by the user.
*   **Modern UI/UX:** Built with a premium Glassmorphism design and Tailwind CSS, featuring a seamless Dark Mode toggle.
*   **Analytics Dashboard:** Visualizes interaction trends and sentiment distribution using Recharts.
*   **Reporting:** Instant PDF export functionality for HCP reports using `html2pdf.js`.

## 🛠 Tech Stack

*   **Frontend:** React (Vite), Redux Toolkit, Tailwind CSS, Recharts, `html2pdf.js`.
*   **Backend:** FastAPI, LangGraph, LangChain, Groq/Llama-based LLMs.
*   **Database:** SQLite for persistent interaction logs.
